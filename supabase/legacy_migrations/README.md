# Archived Lovable-era migrations

These files are retained only as an audit trail. They came from multiple
Lovable database baselines and cannot be replayed as one ordered migration
history because several migrations recreate the same tables.

Do not move these files back into `supabase/migrations` or apply them to
`collectx-prod`. The active migration history is the consolidated,
rehearsed history in `supabase/migrations`.
